Шитье (Tailoring)

Тип: производственная профессия

Другие названия: тейлоринг

Название специалиста: портной, тейлор

Портной умеет шить разнообразные тканые доспехи, формальную одежду (например, рубашки) и сумки.

Ресурсы

В качестве сырья используется ткань, в изобилии падающая с различных человекообразных существ, населяющих мир WoW. Кусочки ткани (например, Льняной материал) преобразуются в полотно (Рулон льняной ткани, соответственно), из которого, в свою очередь, уже и делаются тканые предметы с помощью ниток и красок, покупаемых у торговцев, и других ценных реагентов.

Некоторые высокоуровневые ткани невозможно производить и обрабатывать без специальных условий. Например, для того чтобы создать Луноткань, нужен эльфийский лунный колодец, а чтобы сделать Чароткань, портной должен находиться на территории Netherstorm. Такие ткани можно производить только по одному кусочку с кулдауном в 96 часов.

Чтобы произвести Рулон прочной ткани Пустоты, нужно находиться рядом с волшебным ткацким станком, найти который можно в Шаттрате и Сильвермуне.

Продукты производства

Итак, основными производимыми с помощью шитья предметами являются тканые доспехи. Их огромное множество, но для того, чтобы они хорошо продавались, надо помнить, для кого они предназначены. Самыми востребованными предметами из тканей будут те, которые имеют бонус к интеллекту, выносливости, духу и различным магическим эффектам.

Следующая категория предметов, производимых портными — это сумки (контейнеры) от 6 до 28 слотов, обычные и специализированные (например, для варлоков — Черная сумка теней, для энчантеров — Сумка из огненной чароткани и другие). На низкоуровневые сумки рассчитывать не приходится, они, как правило, не особо дорогие, но на высоких уровнях вместительные сумки становятся отдельной статьей дохода портного.

Формальная одежда не имеет никакого практического смысла в игре. Единственная цель, с которой игрок одевает своего персонажа, например, в смокинг — эстетическая. С другой стороны, такая одежда для своего создания не требуют ничего, кроме ткани, ниток и красок, а, значит, с их помощью можно экономично и просто повышать уровень портного мастерства.

Советы

Больше всего шитье подходит тем классам, которые максимально ограничены в выборе одежды: магам, варлокам и жрецам.

Шитье — единственная производственная профессия, для которой не нужны какие-либо добывающие навыки. Поэтому при выборе второй профессии на начальных уровнях игроку предоставляется свобода, но все же наиболее приемлемой парой шитью традиционно считаются снятие шкур и зачарование.

Если хороший (зеленый) предмет не особо нужный, но выгодный по производным для улучшения навыка профессии, то не торопитесь продавать его вендору. Помните, что такие предметы разбираются на ценные реагенты для зачарования, и куда выгоднее будет выложить их на аукцион. Подходящую цену на такие предметы вам подскажет аддон Auctioneer.

Специализации

При навыке профессии больше 350 и уровне персонажа 60 и выше становятся доступны квесты для профессиональной специализации, выполнив которые портной может выучить уникальные эпические рецепты и получит шанс произвести дополнительное сырье для шитья.

Каждая специализация имеет определенную направленность:

  • Mooncloth для лекарей.
  • Shadoweave для тех, кто пользуется теневой магией и магией мороза.
  • Spellfire для огненных и мистических магов.

Получение материалов.

В отличие от других производственных профессий, портные изготавливают одежду из ткани найденной в вещах убитых гуманоидов. Чем выше уровень противника, тем качественнее ткань может быть обнаружена.

  •  5-17 Льняной материал
  • 16-27 Шерсть
  • 25-40 Шелковый материал
  • 40-50 Магическая ткань
  • 50-55 Руническая ткань
  • 55-60 Ткань скверны
  • 61-70 Ткань Пустоты

Сумки.

Сумки для вещей - идентичны вашему Backpack, от 6 до 18 слотов. Т.к. количество дополнительных сумок ограничено, выгоднее использовать сумки большого объема. Изготовление сумок - неплохой источник дохода, особенно на "молодых" серверах. Изготовление сумок так же неплохой способ развить навык шитья.

Наиболее востребованные сумки:

  • Сумка из рунической ткани - 14 слотов.
  • Сумка из луноткани - 16 слотов.
  • Сумки для ингредиентов. Специальные контейнеры, которые можно использовать исключительно для определенных материалов - трав и реагентов для зачарования, отличаются большей вместимостью, от 16 до 24 слотов.
  • Сумки для камней душ. Используются калдунами для хранения камней-душ (Soulshard), от 20 до 28 слотов.

Ценные рецепты.

Классовые робы, доступны только портным:
  • Одежда истинной веры - роба для жрецов
  • Одеяние верховного мага- роба для магов
  • Одеяние бездны - роба для колдунов
Одежда:
  • Одеяние зимней ночи – отличная роба для фрост-магов, колдунов или жрецов
  • Перчатки из ткани скверны – превосходные перчатки для колдунов и жрецов
  • Инфернальные перчатки – перчатки для огненных-магов
  • Одеяние из луноткани – хорошая роба на начальном этапе игры
  • Одеяние кровавой лозы - бонус от собранного сета получают только портные.
  • Поножи пламенных недр – превосходные штаны для прохождения Molten Core
  • Перчатки волшебства – перчатки для ПвП
  • Пояс верховного мага - ПвП пояс
Сумки для ингредиентов:
  • Сумка из ткани скверны – вполне доступный контейнер для Soul Shards 24 слота
  • Сумка из сердцевинной ткани Скверны – лучшая сумка для камней-душ 28 слотов, требует множества редких материалов.
  • Большая зачарованная сумка– самая вместительная сумка для магических реагентов.
  • Сумка Кенария – лучшая сумка для трав
